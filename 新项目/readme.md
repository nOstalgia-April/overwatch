# 守卫雅典娜（铁坂版本）玩法策划文档

## 1. 核心玩法概述

本模式为基于铁坂地图的多人合作塔防玩法。玩家共同守卫地图中心的核心点，抵御从多个方向涌入的敌人。与传统线性塔防不同，本地图为多通路来敌，强调团队分工、区域控制与节奏协作。

玩法目标为：

* 前期给予爽感与成长体验
* 中期通过运营、波段思考与小Boss挑战决定队伍构筑方向
* 后期通过击败阶段性大Boss解锁成长上限，进入高压循环

游戏胜利条件：完成全波次或击败最终Boss。
游戏失败条件：核心点血量归零或队伍全灭未能及时复活。

---

## 2. 玩家人数与职业结构

本模式推荐 **4 人队伍**，构成稳态战斗循环：

* 查莉娅（坦位 / 阵地维持或破局执行）
* 艾什（远程高压 / 高爆发或空间压制）
* 死神（前线精英处理 / 狂战或阵地压制）
* 芙蕾雅（清杂机动 / 切群或封锁控制）

队伍结构形成：

```
坦位维持 → 输出执行 → 清杂稳定场面 → 前线循环续命
```

---

## 3. 波次与节奏结构

波次以循环节奏为单位划分：

* 小怪波：提供基础成长资源（主资源）
* 精英波：给予队伍爆发与结构检验（副资源来源）
* 小Boss挑战阶段（可选）：提供流派质变资源（稀有资源）
* 大Boss阶段（强制）：检验队伍构筑与运营结果，击败后解锁成长上限（技能二段扩展与收益倍增）

周期示例：

```
1-4: 正常压力波 → 波后主城运营
5: 小Boss挑战可选 → 流派差异显现
6-9: 中压波 → 队伍执行与站位考验
10: 大Boss → 击败进入下一成长阶段
```

---

## 4. 产出系统（成长资源结构）

本模式采用三层级成长资源：

| 资源类型 | 获取来源       | 用途              | 玩家感受        |
| ---- | ---------- | --------------- | ----------- |
| 主资源  | 小怪/精英/波次奖励 | 提升基础属性与技能效率（L1） | 我能活下去       |
| 副资源  | 主城运营奖励/精英波 | 解锁技能分支与操作表现（L2） | 我决定我怎么打     |
| 稀有资源 | 小Boss挑战    | 解锁流派质变（L3）      | 我成为另一个版本的自己 |

大Boss击败后解锁：

* 角色技能二段扩展槽位
* 后续资源获取倍率提升
* 战斗循环上限提升进入后期

---

## 5. 英雄构筑设计（L3 流派 + 大Boss阶段突破）

### 查莉娅（坦位）

流派分歧：

* 护阵核心：为队伍创造可站立的护域，维持前线完整
* 能量行刑：吸收 → 蓄能 → 破甲 → 全队集火
  大Boss阶段突破提供：护盾相位化或裂能循环，保证“顶 → 破 → 顶”循环不间断

### 艾什（主C）

流派分歧：

* 尖锋狙杀：高伤害低频节奏 → 精英斩杀
* 火场压制：持续灼烧与区域削弱 → 控场与线控
  大Boss突破提供：装填链路或灼烧引爆链，保证持续输出节奏成立

### 死神（前线压制）

流派分歧：

* 狂血猎手：贴脸越打越强 → 杀戮驱动续战
* 怨灵压境：区域削弱 → 阵地控制与留场
  大Boss突破提升循环效率与续航强度

### 芙蕾娅（清杂机动）

流派分歧：

* 旋斩风暴：通过切群与连锁突入快速清除怪群
* 地脉震荡：铺设控制线路，削弱群体威胁
  大Boss突破提供转点连锁或震域回响，保证节奏不间断

---

## 6. 主城与运营机制

波次结束 → 进入主城休整：

* L1 基础强化（主资源）
* L2 技能分支与功能表达（副资源）
* 小Boss召唤（选择性挑战获取稀有资源）

运营意义：

* 不运营不会立即输，但会在中期开始逐渐掉队
* 运营质量决定大Boss是否能顺利击败

---

## 7. 难度曲线目标

* 前期：爽感 → 可自由尝试
* 中期：需要决策与协同
* 后期：流派成型者碾压，不成型者崩溃

体验目标：

```
爽 → 思考 → 选择 → 跳跃成长 → 高压成就
```

---

## 8. 结语

本模式设计框架已清晰建立：

* 多路线守点塔防 + 流派化构筑系统
* 清晰职业分工与队伍循环结构
* 玩家选择直接决定队伍命运
